# Odin Recipes

## Description
A basic recipe website built as part of The Odin Project Foundations curriculum. 
The site features a main index page linking to three individual recipe pages.

## Recipes Included
- Lasagna
- Rolex (Ugandan street food)
- Matoke (Ugandan traditional dish)

## What I Learned
- HTML boilerplate structure
- Headings, paragraphs and text elements
- Ordered and unordered lists
- Relative links between pages
- Navigation links between pages
- Adding images with alt and width attributes
- Organizing files and folders properly
- Git and GitHub workflow — committing and pushing changes

## Built With
- HTML5
- Git and GitHub
